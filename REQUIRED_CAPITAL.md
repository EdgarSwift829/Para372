# 372手法改良版EA - 必要資金ガイド

## 最小ロット（0.01）でエントリーするために必要な資金

### 基本的な計算式

```
必要資金 = (最小ロット × SL幅(pips) × 1pipあたりの価値) ÷ リスク%
```

### 通貨ペア別の必要資金目安

#### USD/JPY の場合

**前提条件:**
- 最小ロット: 0.01
- リスク設定: 2%
- ATR倍率: 1.5
- 平均ATR: 約0.5円（50pips）※相場による
- SL幅: ATR × 1.5 = 0.75円 = 75pips

**計算:**
```
0.01ロット × 75pips × 100円/pip ÷ 2% = 37,500円
```

**必要資金の目安: 約4万円～**

### 実際のエントリー例

#### ケース1: 資金10万円、USD/JPY、リスク2%

**シナリオ:**
- 口座残高: 100,000円
- ATR: 50pips
- SL幅: 75pips（ATR × 1.5）

**計算:**
```
リスク金額 = 100,000円 × 2% = 2,000円
0.01ロットのリスク = 75pips × 100円 = 7,500円

計算ロット = 2,000円 ÷ 7,500円 = 0.0267ロット
→ 最小単位（0.01）に丸める = 0.02ロット
```

**エントリー: 0.02ロット** ✅

### ボラティリティによる資金要件の変動

ATRは相場状況で変動します：

| 相場状況 | ATR（USD/JPY） | SL幅 | 必要資金（2%） |
|---------|---------------|------|--------------|
| 低ボラ | 30pips | 45pips | 22,500円 |
| 通常 | 50pips | 75pips | 37,500円 |
| 高ボラ | 80pips | 120pips | 60,000円 |
| 超高ボラ | 100pips | 150pips | 75,000円 |

**重要:** 高ボラ時は必要資金が増えるため、資金が少ないとエントリーできなくなる可能性があります。

### 資金別のエントリー頻度目安

| 資金 | 低ボラ時 | 通常時 | 高ボラ時 |
|------|---------|--------|---------|
| 3万円 | ◯ (0.01) | △ (見送り多) | ✗ (ほぼ見送り) |
| 5万円 | ◯ (0.01) | ◯ (0.01) | △ (0.01) |
| 10万円 | ◯ (0.02) | ◯ (0.02) | ◯ (0.01) |
| 20万円 | ◯ (0.04) | ◯ (0.03) | ◯ (0.02) |

### リスク設定の調整

資金が少ない場合の対処法：

#### オプション1: リスクを下げる
```
RiskPercent = 1.0  // 2%→1%に下げる
```
- メリット: より少ない資金でエントリー可能
- デメリット: 利益の増え方が遅くなる

#### オプション2: ATR倍率を下げる
```
ATR_Multiplier = 1.0  // 1.5→1.0に下げる
```
- メリット: SL幅が狭くなり、より少ない資金でエントリー可能
- デメリット: SLに引っかかりやすくなる

#### オプション3: 資金を追加する（推奨）
- 最も健全な解決方法
- USD/JPYなら最低5万円、推奨10万円

### よくある質問

#### Q1: 最低いくらあればEAを動かせますか？
**A:** USD/JPYなら4万円が最低ラインですが、**10万円以上を推奨**します。4万円だと高ボラ時にエントリーできない場合があります。

#### Q2: 資金が少ないとどうなりますか？
**A:** EAが自動的にエントリーを見送ります。エラーにはなりませんが、トレード機会を逃すことになります。

#### Q3: 資金が増えたら自動的にロットも増えますか？
**A:** はい、自動的に増えます。資金が2倍になれば、エントリーロットもほぼ2倍になります。

#### Q4: 逆に資金が減ったら？
**A:** 自動的にロットも減ります。資金管理が常に2%に保たれるため、破綻リスクは低いです。

#### Q5: 1日に何回くらいトレードしますか？
**A:** 平均して1日0～2回程度です。相場状況によって変動します。
